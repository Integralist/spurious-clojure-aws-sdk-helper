# Spurious Clojure AWS SDK Helper

Inspired by the original [Spurious Ruby AWS SDK Helper](https://github.com/stevenjack/spurious-ruby-awssdk-helper)

## Usage

```bash
$ java -jar spurious-aws-sdk-helper-0.1.0-standalone.jar [args]
```

## Testing locally

```bash
lein pom
lein jar
lein install
```
